Introduction to bootstrap and grid format.

Bootstrap

It is a container working on rows and columns. It is a collection of free and open source tools. It was developed to make life easier by creating all the screen size options into one. This has the folllowing advantages.

  1. Optimized for all screens
  2. Faster and easier
  3. Responsive
  4. Intended which means properly spaced.
  5. Completely free.
It works on a grid format which is as follows.

Grid format

It works on 12 rows, it divides any size of the screen into 12 equal parts and then divides it into equal coloumns as the user likes.It has five sizes

  • Extra small,xs
  • Small,sm
  • Medium,md
  • Large,lg
  • Extra large,xl
